# 🌌 About

**Edemy** is a modern Learning Management System designed with a **"Wow First"** philosophy. It combines elegant design, lightning-fast performance, responsive layouts, glassmorphism, and smooth micro-interactions to deliver a premium experience for both students and educators.

---

# ✨ Features

### 🎓 Student Panel

- 🔍 Smart Course Search
- 📚 Dynamic Course Explorer
- ▶️ YouTube Video Player
- 📈 Progress Tracking
- 🌙 Dark Mode
- 💎 Glassmorphism UI
- ⚡ Smooth Animations
- 📱 Fully Responsive
